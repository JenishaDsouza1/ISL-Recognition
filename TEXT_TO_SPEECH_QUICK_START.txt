┌────────────────────────────────────────────────────────────────┐
│ 🎯 HOW TO USE                                                  │
└────────────────────────────────────────────────────────────────┘

1. Finish signing your sentence
   Final Sentence Card: "hello how are you"

2. Click the "Speak" button
   (Desktop: right side, Mobile: below sentence card)

3. System automatically detects language
   English: No Hindi characters → English voice
   Hindi: Contains Devanagari (नमस्ते) → Hindi voice

4. Browser speaks the text
   🔊 "hello how are you" (in the correct language)

5. Click "Speak" again to stop (if needed)


┌────────────────────────────────────────────────────────────────┐
│ 🌍 LANGUAGE DETECTION                                         │
└────────────────────────────────────────────────────────────────┘

AUTOMATIC - No need to tell the system which language!

English Examples:
  "hello" → Speaks in English ✅
  "how are you" → Speaks in English ✅
  "jenisha is here" → Speaks in English ✅

Hindi Examples:
  "नमस्ते" → Speaks in Hindi ✅
  "कैसे हो" → Speaks in Hindi ✅
  "जेनिशा यहाँ है" → Speaks in Hindi ✅

Mixed Language:
  "hello नमस्ते" → Speaks in Hindi (defaults to Hindi if mixed)


┌────────────────────────────────────────────────────────────────┐
│ 🎙️ HOW IT WORKS                                              │
└────────────────────────────────────────────────────────────────┘

User clicks "Speak"
         ↓
System checks text for Hindi characters (Devanagari)
         ↓
If Hindi characters found → Use Hindi voice (hi-IN)
If no Hindi characters → Use English voice (en-US)
         ↓
System selects best available voice:
  Priority: Google voices > Native voices > First available
         ↓
Browser speaks the text using Web Speech API
         ↓
Toast notification: "✅ Spoken in [Hindi/English]"

┌────────────────────────────────────────────────────────────────┐
│ ✅ WHAT GETS SPOKEN                                           │
└────────────────────────────────────────────────────────────────┘

Priority order (speaks first available):
  1. Full Sentence (if available)
  2. Current Word (if no sentence)
  3. Current Letter (if no word)

Example:
  Sentence: "hello how are you"
  Speaks: "hello how are you" ✅ (full sentence)

  Sentence: "" (empty)
  Word: "hello"
  Speaks: "hello" ✅ (current word)

┌────────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                            │
└────────────────────────────────────────────────────────────────┘

Problem: No sound when clicking "Speak"
Solutions:
  ✓ Check system volume (not muted)
  ✓ Check browser audio settings
  ✓ Try a different browser
  ✓ Refresh the page

Problem: Wrong language voice (Hindi in English or vice versa)
Solutions:
  ✓ Install Hindi language pack:
    Windows: Settings → Language → Add Hindi
    macOS: System Prefs → Spoken Content
    Android: Settings → Text-to-speech → Install Hindi
  ✓ Use Chrome or Edge (best Hindi support)
  ✓ Check if text is correct language

Problem: Speech cuts off
Solutions:
  ✓ Some browsers limit length (~160 characters)
  ✓ Try shorter sentences
  ✓ Check browser console for errors

Problem: "No voice available" error
Solutions:
  ✓ Wait 2-3 seconds for voices to load
  ✓ Refresh the page
  ✓ Install language support
  ✓ Try different browser
