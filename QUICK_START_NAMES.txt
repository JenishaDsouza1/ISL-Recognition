┌────────────────────────────────────────────────────────────────┐
│ 📝 HOW TO ADD YOUR NAMES (Choose One Method)                  │
└────────────────────────────────────────────────────────────────┘

METHOD 1: Edit File (Permanent) ⭐ RECOMMENDED
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Open: src/lib/wordCorrection.ts
2. Add to wordCorrections dictionary:
   
   'wrong_spelling': 'correct_name',
   
   Example:
   'jon': 'john',
   'sara': 'sarah',

3. Save file
4. Test: WordTester.testWords(['jon', 'sara'])


METHOD 2: Browser Console (Testing Only)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Press F12 (open console)
2. Type:
   
   WordTester.quickAdd('wrong', 'correct')
   
   Example:
   WordTester.quickAdd('geisha', 'jenisha')

3. Test: WordTester.testSentence('hi i am geisha')


METHOD 3: Copy from Template (50+ Names Ready!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Open: NAME_CORRECTIONS_TEMPLATE.txt
2. Copy the names you need
3. Paste into: src/lib/wordCorrection.ts
4. Uncomment (remove //) the ones you want
5. Save!


┌────────────────────────────────────────────────────────────────┐
│ 🧪 HOW TO TEST (Browser Console - Press F12)                  │
└────────────────────────────────────────────────────────────────┘

Test common names:
  WordTester.testCommonNames()

Test specific words:
  WordTester.testWords(['geisha', 'sara', 'mike'])

Test a full sentence:
  WordTester.testSentence('hi i am geisha')

Show all corrections:
  WordTester.showAllCorrections()

Discover what to add:
  WordTester.discoverCorrections(['geisha'], ['jenisha'])


┌────────────────────────────────────────────────────────────────┐
│ ✅ HOW TO VERIFY IT'S WORKING                                 │
└────────────────────────────────────────────────────────────────┘

1. Sign a name (e.g., "Jenisha")
2. Backend might send: "geisha"
3. Screen shows: "jenisha" ✅
4. Click "Fix Grammar" button
5. Name stays: "jenisha" ✅
6. Check console for: "Word corrections applied after grammar check"

┌────────────────────────────────────────────────────────────────┐
│ ⚡ IMPORTANT RULES                                            │
└────────────────────────────────────────────────────────────────┘

✅ DO:
  - Use all lowercase in dictionary
  - Add phonetic variations (geisha, geesha, gaisha)
  - Test with "Fix Grammar" button
  - Check console logs

❌ DON'T:
  - Use capital letters in dictionary (won't work!)
  - Forget the comma at the end
  - Skip testing


┌────────────────────────────────────────────────────────────────┐
│ 🎯 HOW IT PROTECTS YOUR NAMES                                │
└────────────────────────────────────────────────────────────────┘

TWO-STAGE PROTECTION:

Stage 1: WebSocket receives from backend
  Backend: "geisha is here"
    ↓
  Corrected: "jenisha is here" ✅

Stage 2: Grammar correction (if you click "Fix Grammar")
  LanguageTool might change: "geisha is here"
    ↓
  Re-corrected: "jenisha is here" ✅

Result: Names ALWAYS correct! 🎉
